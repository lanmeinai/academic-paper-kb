# Skill：写论文辅助

## 触发方式
用户说「写论文辅助」或「帮我写 Related Work」时执行本 Skill。

## 执行流程

### 第一步：获取主题
询问用户：「请描述你的论文主题（一两句话即可）」

如果用户在 config.json 中设置了 research_domain，可以此作为默认上下文。

### 第二步：读取知识库
读取 data/index.json，根据主题关键词匹配相关论文：
- 比对标签、中文标题、核心贡献字段
- 找出相关论文，按年份排序

### 第三步：调用 DeepSeek API 生成框架
将匹配到的论文信息发送给 DeepSeek，要求其输出：

1. **Related Work 推荐结构**（3-5个子章节，每节说明写什么）
2. **每个子章节推荐引用的论文**（从库内论文中选，说明为什么引用）
3. **必引核心论文清单**（被多篇库内论文引用、影响力高的）
4. **研究空白分析**（现有方法的不足，即你的论文切入点）
5. **技术演进时间线**（该方向从早到晚的代表性工作）

### 第四步：输出结果
以结构化中文输出上述内容，方便直接参考写作。

### 第五步：询问是否导出 BibTeX
「是否需要将以上论文导出为 BibTeX 格式？」
