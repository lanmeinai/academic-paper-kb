**Academic Paper KB** 是一个 **AI 驱动的个人学术知识库**——你把论文 PDF 扔进 `inbox/`，Claude Code + DeepSeek 自动完成阅读、分析、分类、归档、建立引用网络。最终在 Obsidian 里呈现一个**活的、可探索的知识图谱**。

> 🧠 它不是"又一个文献管理工具"，而是**把你读过的所有论文变成一个可对话的知识体**。

## 🎪 核心卖点

### 🔥 卖点一：全自动知识提取 —— 你只需要"扔"

你不需要：
- ❌ 手动重命名 PDF
- ❌ 手动填元数据
- ❌ 手动写摘要
- ❌ 手动打标签
- ❌ 手动分类
- ❌ 手动建立引用关系

你需要做的只有一件事：**把 PDF 复制到 `inbox/`，说"处理新论文"。**

### 🔥 卖点二：Obsidian 可视化知识图谱

![知识图谱示意](https://img.shields.io/badge/Graph_View-已连接的知识网络-7c3aed)

- 每个节点 = 一篇论文（节点名包含年份 + 作者 + 关键词）
- 每条连线 = 引用关系（线越粗 = 被引用越多）
- 不同颜色 = 不同研究方向
- 点击节点 → 直接打开完整的中文笔记

**配合 Obsidian 的 Dataview 插件，可以查询任意维度的论文集。** 比如：
```dataview
TABLE 年份, 标签, 核心贡献
FROM "论文笔记"
WHERE contains(标签, "深度学习") AND 年份 > 2022
SORT 年份 DESC
```

### 🔥 卖点三：写论文时当外挂大脑

当你跟 Claude Code 说"帮我写 Related Work"，它会：

1. 📋 **自动梳理你的论文库**——找出所有相关的论文
2. 🗂️ **生成推荐结构**——3-5 个子章节，每个配推荐引用
3. ⭐ **标出必引核心论文**——不会漏掉领域奠基性文献
4. 🕳️ **分析研究空白**——告诉你哪里有 gap，你的论文可以切入
5. 📜 **绘制技术演进时间线**——一目了然这个方向怎么发展起来的

### 🔥 卖点四：领域无关 + 可配置

| 🖥️ 计算机/AI | 🔬 生物/医学 | ⚙️ 工程 | 📊 社科 | 🧪 化学/材料 |
|-------------|------------|--------|--------|------------|
| CV/NLP/RL | 基因组学 | 控制理论 | 经济学 | 催化/合成 |
| 深度学习 | 药物发现 | 信号处理 | 心理学 | 表征分析 |
| 系统/架构 | 临床研究 | 优化算法 | 教育学 | 理论计算 |

编辑 `config.json` 一行即可切换领域，AI 自动调整分析视角。

## 📂 项目结构

```
academic-paper-kb/
│
├── 📥 inbox/                    ← 把新 PDF 扔这里（唯一的入口）
│
├── 📂 papers/                   ← 处理完的 PDF 自动归档
│   ├── 实验研究/
│   ├── 深度学习/
│   └── ...
│
├── 📊 data/
│   └── index.json              ← 所有论文的结构化索引（核心数据库）
│
├── 🧠 vault/                    ← 用 Obsidian 打开这个目录
│   ├── 论文笔记/                ← 每篇论文一个中文笔记
│   ├── 主题索引/                ← 按标签聚合的论文列表
│   └── 作者索引/                ← 按作者聚合的论文列表
│
├── 🎮 skills/                   ← Claude Code Skill 定义
│   ├── 处理新论文/SKILL.md
│   ├── 写论文辅助/SKILL.md
│   └── 导出参考文献/SKILL.md
│
├── 📤 exports/                  ← 导出的 BibTeX 文件
├── ⚙️ config.json               ← 你的研究领域配置
└── 🔑 .env                      ← DeepSeek API Key
```

## 🚀 5 分钟启动

```bash
# 1 分钟：克隆 + 安装
git clone https://github.com/lanmeinai/academic-paper-kb.git
cd academic-paper-kb
pip install -r requirements.txt

# 1 分钟：配置 API Key
cp .env.example .env
# 编辑 .env 填入你的 DeepSeek API Key
# 去 platform.deepseek.com 免费注册即可获得

# 1 分钟：设置研究领域
# 编辑 config.json：
# { "research_domain": "你的领域", "domain_hint": "重点方向" }

# 2 分钟：用 Obsidian 打开 vault/ 目录
# 推荐安装插件：Dataview、Graph Analysis

# 完事！扔 PDF 进 inbox/，对 Claude Code 说"处理新论文"
```

## ❓ FAQ

<details>
<summary><b>Q: 扫描版 PDF（无法选中文字）能用吗？</b></summary>

不能直接处理。扫描版 PDF 没有文字层，PyMuPDF 提取不到。建议用 OCR 工具（Adobe Acrobat / 在线工具）预处理后再放入 inbox。
</details>

<details>
<summary><b>Q: AI 分析结果不满意怎么办？</b></summary>

所有输出都是你可以手动编辑的——笔记是 `.md` 文件，索引是 `data/index.json`。改完即刻生效，Obsidian 关系图谱实时更新。
</details>

<details>
<summary><b>Q: 引用关系识别不全怎么办？</b></summary>

在笔记的"引用了（库内论文）"部分手动添加 `[[论文笔记名]]`。Obsidian 会自动建立双向链接。你手动加一次，以后一直都在。
</details>

<details>
<summary><b>Q: 要钱吗？</b></summary>

框架 MIT 协议永久免费。你需要 DeepSeek API Key（每篇论文分析成本约 ¥0.01-0.05，充 10 块钱用到天荒地老）。
</details>

<details>
<summary><b>Q: 我不是计算机专业的，能用吗？</b></summary>

完全能。编辑 `config.json` 设置你的研究领域即可。AI 自动适配分析视角和标签体系。医学、化学、材料、社科、人文——都能用。
</details>
